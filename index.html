StyleTransfer
StyleTransfer
Web application that utilizes a deep neural network to provide a real-time means of styling user images with various famous artwork styles (e.g. Van Gogh's Starry Night). The style transfer algorithm was taken from the tensorflow_hub library in Python. The frontend was built using HTML, CSS, and Javascript, and backend was built using Flask.
Live Demo!
X-Malaria
X-Malaria: Automated Malaria Diagnosis
Web application that performs malaria diagnosis from thin blood smear images using two deep learning models. A custom-trained You-Only-Look-Once (YOLO) object detection algorithm detects and segments individual red blood cells from the entire blood smear image, and a standard convolutional neural network classifies individual red blood cells as infected or uninfected by malaria parasite. The deep learning models were developed using Keras and Tensorflow in Python. The frontend was built using HTML, CSS, and Javascript, and backend was built using Flask.
Live Demo!
HeartHelper
HeartHelper
Web application that diagnoses risk of heart disease from 13 clinical characteristics (e.g. sex, age, resting blood pressire) using the Gaussian Naive Bayes algorithm, which achieved ~90% diagnostic accuracy. The predictive models was developed using Sci-Kit Learn in Python. The frontend was built using HTML, CSS, and Javascript, and backend was built using Flask.
